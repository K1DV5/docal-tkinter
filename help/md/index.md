# Quick help

__docal__ is a desktop app to create documents that include calculations. To
learn from the beginning, head to the [Tutorial](tutorial.html). If you
want to get detailed information, see the [Reference](reference.html). Here are
some quick points.

* To insert text, begin or end with a space.
* Consecutive text lines and inline equations are put in one paragraph. To separate paragraphs, use an empty line.
* To write calculations, write them just as you would in a calculator
* To insert an inline equation, begin with $
* To insert a block equation, begin with $$
